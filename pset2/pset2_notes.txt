Command line arguments

main accepts two parameters: argc and argv[]

argc = stands for argument count, is an int and is the number of arguments including the command itself
argv = argument vector, each element in arguments

int main(int argc, char * argv[])

{
    for (int i = 0; i < argc; i++) 
        printf("argv[%d] is: %s \n", argc, argv[i]
    return 0;
}
